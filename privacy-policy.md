We collect the following information:
- User Name
- Mobile Number

- User Current Location (to show nearest CNG pump)
- Pump Location Coordinates (for pump management)

## 2. Why We Collect This Data
We collect this data to:
- Create and manage user accounts
- Allow secure login from a single device
- Show nearest CNG pumps
- Manage and display pump locations

We do NOT sell or share your personal data with any third parties.

## 3. Location Access
We use your location only to show nearest CNG pumps.
Location data is not shared with anyone.

## 4. Data Security
We take reasonable security measures to protect your data against unauthorized access.

## 5. Data Sharing

# मराठी आवृत्ती (Marathi Version)

CNG मार्गदर्शक अ‍ॅप तुमच्या गोपनीयतेचा आदर करते. हे Privacy Policy तुमची माहिती आम्ही कशी घेतो, का घेतो आणि कशी वापरतो याबद्दल माहिती देते.

## 1. आम्ही कोणती माहिती गोळा करतो?
आम्ही खालील माहिती घेतो:
- वापरकर्त्याचे नाव  
- मोबाईल नंबर  
- डिव्हाइस आयडी (एका मोबाईलवरच लॉगिन ठेवण्यासाठी)  
- वापरकर्त्याचे सध्याचे लोकेशन (जवळचे CNG पंप दाखवण्यासाठी)  
- CNG पंपचे लोकेशन कोऑर्डिनेट्स  

## 2. ही माहिती का घेतो?
ही माहिती खालील कारणांसाठी वापरली जाते:
- खाते तयार करण्यासाठी व व्यवस्थापनासाठी  
- सुरक्षित लॉगिनसाठी  
- जवळचे CNG पंप दाखवण्यासाठी  
- पंप लोकेशन व्यवस्थापनासाठी  

तुमची माहिती आम्ही कोणत्याही तृतीय पक्षाला विकत नाही किंवा शेअर करत नाही.

## 3. लोकेशन वापर
तुमचे लोकेशन फक्त जवळचे CNG पंप दाखवण्यासाठी वापरले जाते.
हे लोकेशन कोणासोबतही शेअर केले जात नाही.

## 4. डेटा सुरक्षा
तुमचा डेटा सुरक्षित ठेवण्यासाठी आम्ही योग्य सुरक्षा उपाय करतो.

## 5. डेटा शेअरिंग
तुमची वैयक्तिक माहिती खालील ठिकाणी शेअर केली जात नाही:
- जाहिरात कंपन्या  
- तृतीय पक्ष कंपन्या  
- कोणत्याही बाहेरील संस्थेकडे (कायदेशीर गरज असल्याशिवाय)

## 6. वापरकर्त्याचे हक्क
वापरकर्ता खालील गोष्टी मागू शकतो:
